Detector de Humo con Python

07/09/2024

La finalidad de este proyecto es detectar incendios con ayuda de una tarjeta esp8266 de expressif, se aprovecha el uso de las redes sociales para informar sobre los incendios. Con el uso del SDK prporcionado por expressif que principalmente trabaja con FreertOS. Por parte de la computadora que monitoria se utilizo Python para generar una GUI y para programar el bot que revisa continuamente la esp8266.

Se selecciona el sensor mq135 ya que es capaz de detectar humo y Co2, sin embargo para este ejercicion no se conecta, sino solo se generaron valores dentro del mismo script ya que la finalidad de este proyecto era probar la conexion wifi de la placa esp8266.

Como se menciona con anterioridad decidi hacer este proyecto para probar las conexiones a internet, ya que, hoy en dia el internet de las cosas es bastante interesante aparte de que dia a dia aumenta su popularidad y usos. Uso el protocolo de comunicacion MQTT para establecer una conexion con adafruit.io que ofrece soporte para este tipo de proyectos.
